# Supported Formats

UltimaPy works with a number of Ultima Online client file formats:

- `art.mul` / `artidx.mul` – static and land art
- `gumpart.mul` – gump graphics
- `hues.mul` – hue tables
- `tiledata.mul` – land and item tile data
- `Body.def`, `Bodyconv.def` – body mapping and overrides
- `Skills.idx`, `skills.mul` – skills
- `skillgrp.mul` – skill groups
- `fonts.mul` – ASCII fonts
- `verdata.mul` – patch overrides
