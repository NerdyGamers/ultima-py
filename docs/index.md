# UltimaPy

UltimaPy is a Python SDK for reading and working with **Ultima Online** client data files
(`art.mul`, `hues.mul`, `gumpart.mul`, `tiledata.mul`, `skills.mul`, etc).

It provides a programmatic interface to:

- Inspect and decode client resources
- Render art and gumps into `Pillow` images
- Access tiledata, bodies, hues, skills and related metadata
- Build tooling on top of the Ultima Online asset format
